Privacy

Last updated: 2026-04-18

games.khe.ee is a public personal home-server page. No advertising, no user accounts, no visitor behaviour tracking (no analytics).

Cookies

The page itself sets no cookies. Cloudflare's CDN may set its bot management cookie _cf_bm (~30 min lifetime) — considered strictly necessary under EU ePrivacy rules and requires no consent.

Local storage

  • Smart Games (/study/) — game state (level progress, etc.) is stored in the browser's localStorage. The server never sees it.
  • AI Adventure (/adventure/) — game state is stored in the browser's localStorage.

AI Adventure specifics

"AI Adventure" forwards player-entered text to an AI provider API (Anthropic Claude or Google Gemini) to generate the story. Requests pass through a server-side proxy on to the chosen provider.

These providers are US-based and may process request content per their own policies:

  • Anthropic — privacy policy
  • Google Gemini — terms

Advice: do not enter real names, addresses, national ID numbers, or other personal data you would not want to send to a third party.

Hosting and network

  • Server is in Tallinn, on a home LAN.
  • Public access goes through Cloudflare (DNS, tunnel, TLS). Cloudflare can see request metadata (IP, URL, user agent) and acts as the TLS endpoint.
  • Fonts are served from this server, not a third party.
