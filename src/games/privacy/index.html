Privacy

Last updated: 2026-04-26

games.khe.ee is a public personal home-server page. It has no advertising, no user accounts, and no tracking for advertising or profiling.

Analytics

If you allow analytics, Cloudflare Web Analytics collects aggregate usage and performance metrics such as visits, page views, top paths, referrers, countries, browser/device types, and page load timing. It is not used for advertising. Your choice is stored in this browser's localStorage under khe.analyticsConsent.v1.

Cookies

The page itself sets no cookies. Cloudflare's CDN may set its bot management cookie _cf_bm, with a lifetime of about 30 minutes. It is considered strictly necessary under EU ePrivacy rules and requires no consent banner.

Browser local storage

  • The language preference is stored in browser localStorage under app_locale, so the selected language can carry between khe.ee, the launcher, and the games.
  • Smart Games (/study/) stores game state, such as level progress, in browser localStorage. The server never sees it.
  • AI Adventure (/adventure/) stores game state in browser localStorage.

AI Adventure

AI Adventure sends player-entered text and selected game settings through a server-side proxy to an AI provider to generate the next part of the story.

The providers used may be US-based and process request content under their own terms:

  • Anthropic - privacy policy
  • Google Gemini - terms

Advice: do not enter real names, addresses, national ID numbers, or other personal data you would not want to send to a third party.

Hosting and network

  • The server runs on a home LAN in Estonia.
  • Public access goes through Cloudflare (DNS, tunnel, TLS). Cloudflare can see request metadata such as IP address, URL, and user agent, and acts as the TLS endpoint.
  • Launcher fonts are served from this server. Some game apps may load fonts from Google Fonts.
