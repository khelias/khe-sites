Privacy

Last updated: 2026-04-18

This site (khe.ee) is a personal page. It does not use analytics, advertising, or third-party tracking. No personal data is collected or stored on the site itself.

Cookies

The only cookie set on this domain is Cloudflare's bot management cookie _cf_bm, issued automatically by the CDN to tell humans from automated traffic. It is considered strictly necessary under EU ePrivacy rules and expires within 30 minutes. No consent banner is required for this kind of cookie.

Third-party requests

Fonts are served from this server, not from a third party. The page makes no outbound requests to external services as it loads.

Hosting and network

  • Hosted on a private server in Tallinn, Estonia.
  • Public access goes through Cloudflare's edge network (DNS, tunnel, TLS termination). Cloudflare can see request metadata such as IP address, URL, and user agent, and operates as the TLS endpoint for the public route.

Other services on this domain

Subdomains (cloud.khe.ee, photos.khe.ee, vault.khe.ee, etc.) host self-hosted applications used by the site owner and immediate family. They are not open to public registration. Each application has its own data handling — see the upstream project documentation if you have account access.
