Privacy

Last updated: 2026-04-26

This site (khe.ee) is a personal page. If you allow analytics, it uses Cloudflare Web Analytics for aggregate page-view and performance statistics. It does not use advertising, profiling, analytics cookies, or cross-site tracking.

Analytics

If you allow analytics, Cloudflare Web Analytics collects aggregate usage and performance metrics such as visits, page views, top paths, referrers, countries, browser/device types, and page load timing. It is not used for advertising. Your choice is stored in this browser's localStorage under khe.analyticsConsent.v1.

Cookies

The only cookie set on this domain is Cloudflare's bot management cookie _cf_bm, issued automatically by the CDN to tell humans from automated traffic. It is considered strictly necessary under EU ePrivacy rules and expires within 30 minutes. No consent banner is required for this kind of cookie.

Third-party requests

Fonts and the analytics consent script are served from this server. If analytics is allowed, the Cloudflare Web Analytics beacon loads from static.cloudflareinsights.com and sends measurement data to Cloudflare.

Hosting and network

  • Hosted on a private server in Tallinn, Estonia.
  • Public access goes through Cloudflare's edge network (DNS, tunnel, TLS termination). Cloudflare can see request metadata such as IP address, URL, and user agent, and operates as the TLS endpoint for the public route.

Other services on this domain

Subdomains (cloud.khe.ee, photos.khe.ee, vault.khe.ee, etc.) host self-hosted applications used by the site owner and immediate family. They are not open to public registration. Each application has its own data handling.
